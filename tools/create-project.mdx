Usage Examples

# Interactive mode

polynx create

# Direct commands

polynx create fastapi user-service
polynx create ts-package utils
polynx create py-package data-utils
polynx create health-check-provider sonar

# Presets

polynx init vite my-app
polynx init nx

# Help

polynx list
polynx help create
polynx help --ai "how do I add a route?"
